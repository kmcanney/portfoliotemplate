Calibration

Sensors are calibrated using a series of captured images.

The overall process follows:

  1. Install the cameras using the 4-Sensor Reality Capture setup or 9-Sensor Reality Capture setup as directed by your Sony technician. For more information, see Installing the cameras.

  2. Capture the AR marker.

  3. Arrange flags in the target area.

  4. Capture the flags.

  5. Capture the calibration checkerboard as you move it through the target area.

  6. Calibrate captured clips (performed by a remote Sony technician).

  7. Build the 3D model (ply) (performed by a remote Sony technician).

Note

When using the 9-sensor Reality Capture workflow, the calibration procedures will be performed twice. For the first calibration, the sensors will be arranged as shown by the yellow dots in the following image. For the second calibration and content capture, the sensors will be shifted 20 degrees (each sensor will be moved clockwise halfway to the next sensor) as shown by the magenta dots.
